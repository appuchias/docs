# Callout creation

Use `> ` blocks adding `[!type]` to the first line, where `type` is one of the [[Callouts#Supported types | Supported types]]

> [!note]
> Text inside the callout
> > [!tip]
> > Nesting is allowed

# Folding

You can fold and/or unfold callouts using `+` and `-` after the type.

- Unfolded by default

> [!note]+
> Some text

- Folded by default

> [!note]-
> Some text
